# GoSnap 采集 App

- `html-demo/` — 浏览器 HTML 原型（GoSnap V1.0）
- `gosnap-app/` — Expo / React Native 原型

## HTML Demo 在线预览

部署到 GitHub Pages 后，访问：

`https://<your-github-username>.github.io/<repo-name>/`

直达地图页：`https://<your-github-username>.github.io/<repo-name>/?demo=map`

本地预览：

```bash
cd html-demo
python3 -m http.server 8080
```
